# Claude Code ワークスペース

## Why
Claude Code を使って業務・開発・自動化など複数プロジェクトを横断的に進める汎用作業スペース。

## Where
設定 → .claude/settings.json
ルール → .claude/rules.md
メモリ → ~/.claude/projects/

## What
.claude/settings.json → ツール自動承認の許可リスト
.claude/rules.md → 認証情報の扱いや文字コードに関するルール
credentials/ → 認証情報ファイルの格納フォルダ（コードへの直接記述禁止）
DOCS/ → 詳細ドキュメント（必要時のみAIが読み込む）
